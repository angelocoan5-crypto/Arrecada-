# 💰 MetaColetiva

MetaColetiva é uma plataforma simples de crowdfunding onde usuários podem criar projetos e arrecadar dinheiro online.

O objetivo do projeto é permitir que qualquer pessoa publique sua ideia, defina uma meta financeira e receba contribuições de apoiadores.

---

# 🚀 Funcionalidades

✔️ Login simples  
✔️ Criação de projetos  
✔️ Metas financeiras  
✔️ Barra de progresso  
✔️ Sistema de doações simuladas  
✔️ Comissão automática para a plataforma  
✔️ Armazenamento local (LocalStorage)

---

# 🖥️ Demonstração

A plataforma permite:

1. Criar um usuário (login simples)
2. Criar projetos com:
   - Título
   - Descrição
   - Meta financeira
3. Visualizar projetos criados
4. Realizar doações simuladas
5. Acompanhar o progresso das metas
